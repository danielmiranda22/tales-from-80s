# Tales From 80s

A colorful, 80s-inspired dark theme for Visual Studio — deep blue-violet backgrounds, a cool electric-blue accent, and warm syntax highlights tuned for comfortable long coding sessions.

Based on [one-monokai-80s](https://github.com/marcelo-mason/one-monokai-80s) by marcelo-mason.

## Palette

| Role | Hex | | Role | Hex |
|---|---|---|---|---|
| Background | `#0E101F` | | Keyword | `#61A2EF` |
| Editor | `#1F2237` | | String | `#C9B999` |
| Foreground | `#A6B1C9` | | Comment | `#5D688A` |
| Accent | `#6075ED` | | Number | `#D19A66` |
| Selection | `#303656` | | Class / Method | `#56B6C2` / `#D491CF` |
